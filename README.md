> Molecular modeling javascript framework


## Getting Started

Install the module with: `npm install chemjs`

```js
var chemjs = require('chemjs');
chemjs.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g chemjs
$ chemjs --help
$ chemjs --version
```


```sh
# creates a browser.js
$ grunt browserify
```
